# covid_reminder_line_bot

COVID19の感染者数通知を行うLINE botです

## Requirements

- python 3.9.6

## Installation

```sh
# リポジトリをクローン
git clone git@github.com:Futaba-Kosuke/covid_reminder_line_bot.git

# パッケージを導入
poetry install
```

## Usage

```sh
# 仮想環境に入る
poetry shell

# Start the uvicorn server
python main.py
```
